## EMNLP '26  
- The released artifacts are intended only for research and evaluation of programming knowledge tracing models.  
- The released data are derived from anonymized CSEDM programming logs. We do not include personally identifying student information, and the released artifacts contain problem texts, code submissions, labels, and derived skill mappings used for research purposes.
