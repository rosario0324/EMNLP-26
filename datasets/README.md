# Dataset Package

This folder contains the original CSEDM source distributions used as input to
the packaged preprocessing code.

| Submission folder | Contents |
| --- | --- |
| `CSEDM_S19_original/` | S19 raw main table, code states, link table, and supplied split files |
| `CSEDM_F19_original/` | F19 raw main table, code states, link table, and supplied split files |
| `CSEDM_CodeWorkout_original/` | CodeWorkout raw main table and link tables |

The data folder contains source data only. Processed task tables,
cross-validation splits, embeddings, and model outputs are not included.

## Redistribution Note

CSEDM is an externally distributed dataset. Confirm the dataset license and
the submission venue's artifact policy before uploading these raw files to a
public or reviewer-accessible location.
